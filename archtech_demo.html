Development Roadmap

Structured phases from foundation setup to full traceability automation.

Phase 1: Foundation & Setup
Set up LangChain, LLaMA 70B endpoints. Establish synthetic dataset of TechSpec, ICD, and IRS examples with air-gapped environments.
Phase 2: RAG Engine Development
Implement PDF parsing via PyPDF2 and spaCy. Integrate sentence-transformers for vector embeddings stored in FAISS. Fine-tune ArchTech_ai 1B model.
Phase 3: Frontend & Dashboarding
Build role-based Dashboards using React, TailwindCSS, and Shadcn UI. Establish WebSocket hooks for real-time document sync.
Phase 4: Traceability & Automation (Current Phase)
Deploy intelligent AI agents (Document Generator, Change Propagator). Construct bidirectional Artifact linking and validation engine.
